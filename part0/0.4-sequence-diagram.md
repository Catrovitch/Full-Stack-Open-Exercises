# Webpage Button Press Sequence Diagram

```mermaid
sequenceDiagram
    participant Browser
    participant Server

    Browser->>Server: Press Button

    Note over Server: Form submit event\nHTTP POST request to "new_note"
    Browser->>Server: HTTP POST to "new_note"
    Server->>Browser: HTTP Status 302 (Redirect)\nLocation: "notes"

    Note over Browser, Server: Browser reloads Notes page\nThree more HTTP requests
    Browser->>Server: Fetch main.css
    Server-->>Browser: main.css file
    Browser->>Server: Fetch main.js
    Server-->>Browser: main.js file
    Browser->>Server: Fetch data.json
    Server-->>Browser: raw data of notes
